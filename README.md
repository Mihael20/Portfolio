# Mihael Kusibojoski — Personal Portfolio

A modern, responsive personal portfolio website built with Next.js, showcasing my skills, projects, and experience as a Full Stack Developer.

## 🛠️ Tech Stack

- **Framework** — Next.js 16
- **Language** — TypeScript
- **Styling** — Tailwind CSS, SCSS
- **Animations** — Framer Motion
- **Icons** — FontAwesome
- **Font** — Poppins (Google Fonts)

## 📌 Sections

- **About** — Introduction and overview
- **Services** — What I offer
- **Experiences** — Work history and internships
- **Skills** — Technologies I work with
- **Projects** — Recent works including Coffee Shop App, Fullstack Spring Java, SnapLogic, and DevOps Intern Project
- **Contact** — Links to GitHub, LinkedIn, Email, Instagram, and Telegram
- **Payment** — PayPal payment integration

## 📂 Project Structure

```
src/
├── app/             # Next.js app router
├── components/      # Reusable UI components
├── data/            # Static data (projects, skills, social links)
├── constants/       # App constants and strings
└── types/           # TypeScript types
```

## 🏃 Running Locally

### Prerequisites
- Node.js 18+
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Mihael20/Portfolio

# Navigate to project folder
cd Portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:4000](http://localhost:4000) in your browser.
or open https://portfolio-one-jet-5nl5ka2r2e.vercel.app/ 

## 📦 Build for Production

```bash
npm run build
npm start
```
